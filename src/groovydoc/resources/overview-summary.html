Plan elements

  • plan - defines test plan
  • arguments - defines user variables element inside a test plan
  • argument - defines single user variable

Group elements

  • group - defines users group ('Thread Group')

Config elements

  • authorizations - defines 'Authorization' config element
  • authorization - defines single authorization entry
  • cache - defines 'HTTP Cache Manager' config element
  • cookies - defines 'HTTP Cookie Manager' config element
  • cookie - defines single cookie
  • counter - defines 'Counter' config element
  • csv - defines 'CSV Data Set Config' config element
  • defaults - defines 'HTTP Request Defaults' config element
  • headers - defines 'HTTP Header Manager' config element
  • header - defines single header
  • login - defines 'Login Config Element' config element
  • random - defines 'Random Variable' config element
  • variables - defines 'User Defined Variables' config element
  • variable - defines single user variable
  • body - defines body for 'HTTP Request' sampler
  • insert - defines insert element for external script
  • params - defines parameters for 'HTTP Request' sampler
  • param - defines single parameter for 'HTTP Request' sampler

Controller elements

  • loop - defines 'Critical Section' controller
  • execute - defines generic execution controller (includes 'If', 'While', 'Once Only' and others)
  • for_each - defines 'ForEach' controller
  • include - defines 'Include' controller
  • loop - defines 'Loop' controller
  • simple - defines 'Simple' controller
  • transaction - defines 'Transaction' controller

Sampler elements

  • ajp - defines 'AJP Request' sampler
  • debug - defines 'Debug' sampler
  • flow - defines 'Flow Control Action' sampler
  • http - defines 'HTTP Request' sampler
  • params - defines parameters for 'HTTP Request' sampler
  • param - defines single parameter for 'HTTP Request' sampler
  • body - defines body for 'HTTP Request' sampler
  • jsrsampler - defines 'JSR223' sampler

Assertion elements

  • assert_duration - defines 'Duration Assertion' element
  • assert_json - defines 'JSON Assertion' element
  • jsrassertion - defines 'JSR223 Assertion' element
  • assert_md5hex - defines 'MD5Hex Assertion' element
  • assert_response - defines 'Response Assertion' element
  • assert_size - defines 'Size Assertion' element
  • assert_xpath - defines 'XPath Assertion' element
  • check_response - defines simplified version of response assertions
  • check_request - defines simplified version of request assertions
  • check_size - defines simplified version of 'Size Assertion' element

  • extract_css - defines 'CSS Selector Extractor' element
